AI Fire & Smoke Detection System

Real-Time Fire Monitoring using Deep Learning & YOLO Object Detection

Project Overview

This project uses Artificial Intelligence and Deep Learning techniques to detect fire and smoke at the earliest stage inside rooms using CCTV cameras. The system applies the YOLO (You Only Look Once) object detection model to continuously monitor video streams and identify fire hazards instantly.

Key Features
  • Real-time fire detection
  • Smoke identification using YOLO model
  • Early alert generation
  • Admin monitoring dashboard
  • Web-based monitoring system

How Detection Works

Camera footage is analyzed using Deep Learning algorithms. The YOLO model detects fire and smoke objects in each video frame. When fire probability crosses a threshold, the system generates an alert to the administrator for immediate action.
